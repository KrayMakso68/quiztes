О программе

Программа разработана как подмодуль обучающей программы по дисциплине Теория Электросвязи, кафедра 43.

Руководство пользователя

Программа предоставляет простой интерфейс, удобный для использования на любом виде устройств.

1. Введите свою Фамилию и Имя в корректном формате через пробел, а также номер учебной группы.
2. Выберите вид теста, либо необходимые темы, если выбран пункт "Свой тест".
3. Введите необходимое количество вопросов.

Количество вопросов не может быть меньше количества выбранных тем!

1. Внимательно прочитайте вопрос и варианты ответа.
2. Выберите подходящий вариант ответа.
3. Нажмите кнопку "Ответить".

Если ответить без выбора варианта ответа, то вопрос будет считаться отвеченным неверно!
После нажатия вы увидите результат своего ответа.
Чтобы перейти к следующему вопросу, нажмите кнопку "Следующий вопрос".
Если вопрос послений, появится кнопка "Завершить тест".

Вы можете в любой момент закончить тест, нажав на кнопку "Завершить досрочно"!

При завершении Вы попадаете на страницу результатов.
В зависомости от результата страница меняет свой оттенок:
  • Красный - '2'
  • Жёлтый - '3'
  • Синий - '4'
  • Зелёный - '5'
Есть есть вопросы на которые был дан неверный ответ, то с ними можно ознакомиться в таблице "Неверно".
